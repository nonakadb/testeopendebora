# testeopendebora
Este projeto está relacionado o TesteOpen - que se trata da criação de um site em Wordpress e vinculá-lo ao GitHub.
Ele foi feito diretamente no WordPress.

Para ter certeza, que o site está funcionando foi feito o upload dos seguintes arquivos:

1) htdocs.rar - arquvio que contém toda estrutura do meu projeto (pasta www... e outros);
2) Assets - uma pasta que contém as imagens (assets);
3) elementor.json - arquivo JSON com configurações específicas do plugin Elementor.
